STEPS TO USE THE WEBSITE

Garbage Collection

The user gathers all waste at a single location and captures an image of the collected waste.

Image Upload

The user uploads the image to the platform via the site’s interface.

AI-Based Waste Segmentation & Valuation

The AI model processes the uploaded image, identifies and segments the recyclable items, and generates an estimated value for the waste based on market rates

Selling Option

After receiving the estimated price, the user can choose to sell the recyclable waste by clicking the "Sell" button

Waste Collection

Once the user confirms the sale, a local waste collector is notified, who then collects the waste from the user’s location.

Earn Money

The collected waste is sold to a recycling facility, and both the user and the waste collector earn money from the transaction.
